# Plugin Swag v2

Плагин Swag v2 - это плагин для брутальной мотивации

Что делает плагин?

Плагин Swag v2 отправляет мощные, мотивационные и брутальные фразы по команде .swag в чатах. Вместо обычных шуток и мемов вы получаете слова, которые заряжают на действие, преодоление трудностей и достижение целей.

Нововведение:
1. Полная замена фраз — все 25 стандартных фраз изменены с юмористических на брутальные и мотивационные
2. Удалена спам функция (т.е плагин больше не поддерживает команды .swagspam и .stop)
3. Обновленный стиль — изменен префикс сообщений на "⚔️ BRUTAL MODE ACTIVATED"
4. Упрощенный интерфейс — убраны настройки задержки и количества сообщений
5. Фокус на качестве — одна команда, одна идеальная фраза без лишнего шума

Описание плагина:
Плагин Swag v2 — это минималистичный плагин для отправки мотивационных сообщений в брутальном стиле. Плагин содержит коллекцию из 25+ готовых фраз, каждая из которых начинается с мощного префикса "⚔️ BRUTAL MODE ACTIVATED" и содержит краткое, но сильное высказывание о дисциплине, силе воли, преодолении и достижениях.

Основные характеристики:
1. Единственная команда .swag — отправляет случайную брутальную фразу
2. Гибкие настройки — возможность добавлять собственные фразы и управлять их списком
3. Интуитивный интерфейс — простые настройки через стандартное меню плагинов
4. Стабильная работа — совместимость с последними версиями exteragram и ayugram 

Плагин идеально подходит для мотивационных чатов, групп по саморазвитию, спортивных сообществ или просто для тех, кто предпочитает серьезный настрой вместо легкомысленных развлечений.
